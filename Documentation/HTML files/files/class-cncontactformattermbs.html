Function: The CNContactFormatter class defines the different formatting styles for contacts.
Example: 
dim m as new CNMutableContactMBS
m.givenName = "Bob"
m.familyName = "Miller"

dim f as new CNContactFormatterMBS
dim style as Integer = CNContactFormatterMBS.CNContactFormatterStyleFullName
MsgBox CNContactFormatterMBS.stringFromContact(m, style)
Notes: This class handles international ordering and delimiting for the contact name components. When formatting many contacts, create an instance of this class and use the instance methods; otherwise use the class methods.
