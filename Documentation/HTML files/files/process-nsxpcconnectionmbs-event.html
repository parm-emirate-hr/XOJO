NSXPCConnectionMBS.InterruptionHandler
Type Topic Plugin Version macOS Windows Linux iOS Targets
event Process MBS MacFrameworks Plugin 14.2 ✅ Yes ❌ No ❌ No ❌ No
Function: The event is called if the remote process exits or crashes.
Notes: It may be possible to re-establish the connection by simply sending another message. The handler is invoked on the same queue as reply messages and other handlers, and it is always executed after any other messages or reply block handlers (except for the invalidation handler).

Some examples using this event:

  • /MacFrameworks/NSXPC/Master
NSXPCConnectionMBS.InvalidationHandler
Type Topic Plugin Version macOS Windows Linux iOS Targets
event Process MBS MacFrameworks Plugin 14.2 ✅ Yes ❌ No ❌ No ❌ No
Function: This event is invoked on the same queue as reply messages and other handlers, and is always executed last (after the interruption handler, if required).
Notes: You may not send messages over the connection from within an invalidation handler block.
