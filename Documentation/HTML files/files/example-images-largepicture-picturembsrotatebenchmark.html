EventHandler Function NewPictureMBS(Width as integer, Height as integer, ImageFormat as integer) As PictureMBS window1.Title=str(Width)+" x "+str(Height) dim file as FolderItem dim size as integer n=n+1 // you may need to change file paths if TargetLinux then file=GetFolderItem("/tmp"+str(n)+".test",folderitem.PathTypeShell) elseif TargetWin32 then file=SpecialFolder.desktop.Child("tmp"+str(n)+".test") else file=findvolume("Temp").child("tmp"+str(n)+".test") end if // size of the window: size=100*1024*1024 Return new PictureMBS(Width, Height, ImageFormat, size, file) End EventHandler
