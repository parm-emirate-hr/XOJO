Function: The plugin class which provides parameters for formatting numbers, especially currency values.
Example: 
dim l as LocaleMBS = LocaleMBS.Locale("de_DE")
MsgBox l.CurrencySymbol
Notes: This is an abstract class. You can't create an instance, but you can get one from various plugin functions.
