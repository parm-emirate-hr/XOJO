Function: The class to represent a DNS-SD TXT record.
Example: 
dim t as new TXTRecordMBS

call t.SetValue "Hello", "World"

MsgBox t.Bytes // show complete record
MsgBox t.Value("Hello") // lookup value
Notes: The constructor creates an empty TXTRecord where you can add values using TXTRecordSetValue.
