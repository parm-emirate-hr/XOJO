DynaPDFMBS.YofCMYK(CMYK as UInt32) as Integer
Type Topic Plugin Version macOS Windows Linux iOS Targets
shared method DynaPDF MBS DynaPDF Plugin 10.1 ✅ Yes ✅ Yes ✅ Yes ✅ Yes All
Function: Returns Y of a CMYK color value.
Example: 
dim cmyk as Integer = DynaPDFMBS.CMYK(1,2,3,4)
MsgBox str(DynaPDFMBS.YofCMYK(cmyk)) // shows 3
Notes: Returns different values on BigEndian and LittleEndian systems.

Some examples using this method:

  • /DynaPDF/DynaPDF Makros
  • /DynaPDF/PDF Color Analyze
  • /DynaPDF/PDF Color Analyze per Page
