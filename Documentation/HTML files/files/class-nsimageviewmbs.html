class Cocoa Controls MBS MacControls Plugin 10.0 ✅ Yes ❌ No ❌ No ❌ No Desktop only
Function: An NSImageView object displays a single image from an NSImage object in a frame and can optionally allow a user to drag an image to it.
Example: 
ImageWell1.NSImageViewMBS.alphaValue = 0.5
Notes:
You can embed this view in a CustomNSViewMBS to get more events for mouse and keyboard.
