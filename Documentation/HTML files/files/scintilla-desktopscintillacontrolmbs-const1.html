DesktopScintillaControlMBS.kCharacterSourceTentativeInput = 1
Type Topic Plugin Version
const Scintilla MBS Scintilla Plugin 22.0
Function: One of the character source constants.
Notes: Tentative input characters. They are used by IME (inline mode, see IMEInteraction) to composite final string, normally different from final composited string (which is the string that has been truly added into current document), and may be withdrawn when the user cancels typing (e.g. by pressing Esc key). Some system (at least Cocoa) also use tentative input for non-IME features like using dead key to composite diacritical marks (grave accent, etc.). These characters are not added to macro recording. Most applications can simply ignore the notification when this value is set.
