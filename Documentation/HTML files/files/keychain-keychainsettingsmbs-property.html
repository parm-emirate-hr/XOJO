Type Topic Plugin Version macOS Windows Linux iOS Targets
property Keychain MBS MacClassic Plugin 13.1 ✅ Yes ❌ No ❌ No ✅ Yes All
Function: An unsigned 32-bit integer representing the number of seconds before the keychain locks. If you set useLockInterval to false, set lockInterval to INT_MAX to indicate that the keychain never locks.
Example:
