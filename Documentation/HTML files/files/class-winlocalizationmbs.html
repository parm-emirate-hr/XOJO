class Currency, Date and Time Format MBS Win Plugin 3.3 ❌ No ✅ Yes ❌ No ❌ No Desktop, Console & Web
Function: A class for details about the current localization
Example: 
dim l as WinLocalizationMBS
l=new WinLocalizationMBS
Notes:
This class has a constructor to specify whether you want unicode or not.

You may write yourself functions to acquire the details you need in your application from the Windows and from the Mac classes.
Also you may consider to update your values as the user may change them while your application is running.
