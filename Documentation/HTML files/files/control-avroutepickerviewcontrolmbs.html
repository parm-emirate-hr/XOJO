Button States

Constant Value Description
StateActive 2 Active state of the picker. The picker has this state when AirPlay is active.
StateActiveHighlighted 3 Highlighted state of the active picker. The picker has this state when it is highlighted and AirPlay is active.
StateNormal 0 Normal or default state of the picker.
StateNormalHighlighted 1 Highlighted state of the picker. The picker has this state when a mouse-down event occurs inside the button. It loses this highlight when a mouse-up event occurs.
