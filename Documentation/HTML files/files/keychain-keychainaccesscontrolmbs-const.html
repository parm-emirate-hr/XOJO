KeychainAccessControlMBS.kSecAccessControlBiometryAny = 2
Type Topic Plugin Version
const Keychain MBS MacClassic Plugin 18.5
Function: One of the control flags.
Notes: Touch ID (any finger) or Face ID. Touch ID or Face ID must be available. With Touch ID at least one finger must be enrolled. With Face ID user has to be enrolled.
