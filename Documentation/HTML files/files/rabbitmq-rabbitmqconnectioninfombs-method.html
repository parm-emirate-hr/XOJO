RabbitMQConnectionInfoMBS.Constructor(URL as String)
Type Topic Plugin Version macOS Windows Linux iOS Targets
method RabbitMQ MBS RabbitMQ Plugin 21.2 ✅ Yes ✅ Yes ✅ Yes ✅ Yes All
Function: Parse a connection URL.
Notes:
An amqp connection url takes the form:

amqp://[$USERNAME[:$PASSWORD]@]$HOST[:$PORT]/[$VHOST]

Examples:
amqp://guest:guest@localhost:5672//
amqp://guest:guest@localhost/myvhost

Any missing parts of the URL will be set to the defaults specified in other Constructor. For amqps: URLs the default port will be set to 5671 instead of 5672 for non-SSL URLs.

We raise UnsupportedOperationException if parsing failes.
