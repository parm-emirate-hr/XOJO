Type Topic Plugin Version macOS Windows Linux iOS Targets
shared method Encryption and Hash MBS MacOSX Plugin 11.3 ✅ Yes ❌ No ❌ No ✅ Yes All
Function: One-shot hash calculation with SHA-256.
Notes: Simply pass in a memoryblock with data and the size in bytes and receive digest as memoryblock. If you need to show digest to human, please use EncodingToHexMBS function.
