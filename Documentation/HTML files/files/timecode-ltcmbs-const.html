LTCMBS.LTC_BGF_DONT_TOUCH = 4
Type Topic Plugin Version
const Timecode MBS Tools Plugin 20.1
Function: One of the encoder flags.
Notes: Encoder init or re-init does not touch the BGF bits (initial values after initialization is zero)
LTCMBS.LTC_FRAME_BIT_COUNT = 80
Type Topic Plugin Version
const Timecode MBS Tools Plugin 20.1
Function: The bit count for a raw 80 bit LTC frame
LTCMBS.LTC_NO_PARITY = 8
Type Topic Plugin Version
const Timecode MBS Tools Plugin 20.1
Function: One of the encoder flags.
Notes: Parity bit is left untouched when setting or in/decrementing the encoder frame-number
LTCMBS.LTC_TC_CLOCK = 2
Type Topic Plugin Version
const Timecode MBS Tools Plugin 20.1
Function: One of the frame <> timecode conversion flags.
Notes: The Timecode is wall-clock aka freerun. This also sets BGF1 (unless LTC_BGF_DONT_TOUCH is given)
