class GraphicsMagick MBS GraphicsMagick Plugin 14.1 ✅ Yes ✅ Yes ✅ Yes ✅ Yes All
Function: The class for an array of images in GraphicsMagick.
Example: 
// extract all layers of photoshop file
dim file as FolderItem = SpecialFolder.Desktop.Child("test.psd")
dim images as new GM16ImageArrayMBS

images.readImages(file.NativePath)

dim c as Integer = images.size
for i as Integer = 0 to c-1
  dim image as GM16ImageMBS = images.Image(i)
  file = SpecialFolder.Desktop.Child(image.FileName+"."+str(i)+".png")
  image.write(file)
next
Notes: Can be used to assemble/disassemble gif images.
