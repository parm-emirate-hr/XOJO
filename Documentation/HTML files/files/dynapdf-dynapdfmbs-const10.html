Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 15.0
Function: One of the check flag constants for CheckConformance.

DynaPDFMBS.kcoRepairDamagedImages = &h02000000
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 18.2
Function: One of the check flag constants for CheckConformance.
Notes: Meaningful only if coCheckImages is set. If set, try to recompress a damaged image. The new image is maybe incomplete but error free. This flag can be combined with coDeleteDamagedImages to delete the image if recompression fails.
