Function: A CKNotificationID object uniquely identifies a push notification sent from a container.
Notes:
You do not create notification IDs directly. The server creates them when it creates the CKNotification objects that correspond to the push notifications sent to your app. You can compare two IDs using the isEqual method to determine if two notification objects are the same. This class defines no methods or properties.
This is an abstract class. You can't create an instance, but you can get one from various plugin functions.
