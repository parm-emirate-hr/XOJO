class Cocoa Text MBS MacBase Plugin 22.0 ✅ Yes ❌ No ❌ No ❌ No Desktop, Console & Web
Function: A block of text laid out in a subregion of the text container.
Notes: A text block appears as an attribute of a paragraph, and as part of the paragraph style. The most important subclass of NSTextBlockMBS is NSTextTableBlockMBS, which represents a block of text that appears as a cell in a table. The table itself is a NSTextTable object. All NSTextBlockMBS objects reference this table, which controls their sizing and positioning.
