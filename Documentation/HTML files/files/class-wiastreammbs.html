Function: The WIAStreamMBS interface lets you read and write data to stream objects.
Example: 
dim ItemName as string = "Hello.jpg"
dim f as FolderItem = SpecialFolder.Desktop.Child(ItemName)
dim s as new WIAStreamMBS(WIAStreamMBS.kModeWrite + WIAStreamMBS.kModeCreate, f)
Notes: Stream objects contain the data in a structured storage object, where storages provide the structure. Simple data can be written directly to a stream but, most frequently, streams are elements nested within a storage object. They are similar to standard files.
