Function: The class for a format.
Example: 
dim book as XLBookMBS // your book
// add font
dim boldFont as XLFontMBS = book.addFont()
boldFont.StrikeOut = true

// add format
dim boldFormat as XLFormatMBS = book.addFormat
boldFormat.font = boldFont

// write on a cell in first sheet
sheet = book.Sheet(0)
call sheet.WriteString(1, 1, "Sales Receipt", boldFormat)
Notes: This is an abstract class. You can't create an instance, but you can get one from various plugin functions.
