ScintillaControlMBS.kModificationFlagsStartAction = &h2000
Type Topic Plugin Version
const Scintilla MBS Scintilla Plugin 22.0
Function: One of the modifications flags.
Notes: This is set on a kModificationFlagsUser action when it is the first or only step in an undo transaction. This can be used to integrate the Scintilla undo stack with an undo stack in the container application by adding a Scintilla action to the container's stack for the currently opened container transaction or to open a new container transaction if there is no open container transaction.
