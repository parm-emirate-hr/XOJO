class CloudKit MBS Mac64bit Plugin 16.5 ✅ Yes ❌ No ❌ No ✅ Yes All
Function: A CKNotificationInfo object specifies the push notification data that the server sends to your app when a subscription finds a match.
Notes:
When configuring a CKSubscriptionMBS object, create one of these objects and use it to specify the type of push notifications you want generated when the subscription’s trigger condition is met. You can provide a message for an alert panel, information about the sounds to play, and information about whether the app should be badged. You can also ask the server to provide information about the record that triggered the notification.
