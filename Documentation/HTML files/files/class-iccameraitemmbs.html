Function: ICCameraItem is an abstract class that represents an item in an ICCameraDevice object.
Notes:
ICCameraDevice object creates instances of two concrete subclasses of ICCameraItem: ICCameraFolder and ICCameraFile.
This is an abstract class. You can't create an instance, but you can get one from various plugin functions.
