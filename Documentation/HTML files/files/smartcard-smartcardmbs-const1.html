SmartCardMBS.kAttributeProtocolTypes = 196896
Type Topic Plugin Version
const SmartCard MBS Tools Plugin 16.3
Function: One of the attribute ID constants.
Notes:
Integer encoded as 0x0rrrpppp where rrr is RFU and should be 0x000.
pppp encodes the supported protocol types. A '1' in a given bit position indicates support for the associated ISO protocol, so if bits zero and one are set, both T=0 and T=1 protocols are supported.
