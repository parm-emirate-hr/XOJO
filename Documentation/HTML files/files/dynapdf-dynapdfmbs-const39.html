DynaPDFMBS.kspcDontAddMargins = 2
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 13.4
Function: One of the constants for ConvertEMFSpool.
Notes: If set, the page format is calculated from the EMF files as is. The current page format is not used to calculate margins which are maybe required. Note that the parameters LeftMargin and TopMargin will still be considered.
DynaPDFMBS.kspcFlushPages = 8
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 13.4
Function: One of the constants for ConvertEMFSpool.
Notes: If set, the function writes every finish page directly to the output file to reduce the memory usage. This flag is meaningful only if the PDF file is not created in memory. Note also that it is not possible to access already flushed pages again with EditPage().
DynaPDFMBS.kspcIgnorePaperFormat = 1
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 13.4
Function: One of the constants for ConvertEMFSpool.
Notes: If set, the current page format is used as is for the entire spool file.
DynaPDFMBS.kspcLoadSpoolFontsOnly = 4
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 13.4
Function: One of the constants for ConvertEMFSpool.
Notes: If set, only embedded fonts will be loaded. The EMF files must be converted with the flag mfIgnoreEmbFonts in this case. This flag can be useful if you want to use your own code to convert the EMF files of the spool file.
