CGContextMBS.AddArc(x as Double, y as Double, radius as Double, startangle as Double, endangle as Double, clockwise as boolean)
Type Topic Plugin Version macOS Windows Linux iOS Targets
method CoreGraphics MBS MacCG Plugin 2.7 ✅ Yes ❌ No ❌ No ✅ Yes All
Function: Add an arc to the current path.
Example: 
const r=100.0
dim c as CGContextMBS
// draws a circle with radius r
c.BeginPath
c.SetLineWidth 5
c.SetGrayStrokeColor 0,1
c.AddArc 250,150,r,0,360,false
c.StrokePath
Notes: Add an arc of a circle to the context's path, possibly preceded by a straight line segment. '(x, y)' is the center of the arc; 'radius' is its radius; 'startAngle' is the angle to the first endpoint of the arc; 'endAngle' is the angle to the second endpoint of the arc; and 'clockwise' is true if the arc is to be drawn clockwise, false otherwise. 'startAngle' and 'endAngle' are measured in radians.
