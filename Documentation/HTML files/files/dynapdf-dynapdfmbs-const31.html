DynaPDFMBS.kpfDecomprAllImages = 2
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 8.0
Function: One of the parser flags.
Notes: This flag causes that all image formats will be decompressed with the exception of JBIG2 compressed images. If this flag is not set, images which are already stored in a valid image file format are returned as is. This is the case for RGB JPEG and JPEG2000 compressed images. If you want to extract the images of a PDF file this flag should be absent!
