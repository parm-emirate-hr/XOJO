NSSharingServiceMBS.sharingServicesForItems(items as NSSharingServiceItemsMBS) as NSSharingServiceMBS()
Type Topic Plugin Version macOS Windows Linux iOS Targets
shared method Social MBS MacFrameworks Plugin 12.3 ✅ Yes ❌ No ❌ No ❌ No Desktop only
Function: Returns a list of sharing services which could share all the provided items together.
Example: 
// get an image
dim logo as Picture = LogoMBS(500)
Dim nsi as new NSImageMBS( logo )

// make items object
Dim nshi as new NSSharingServiceItemsMBS
nshi.AddImage nsi

// query services which are supported
Dim ssl(-1) as NSSharingServiceMBS = NSSharingServiceMBS.sharingServicesForItems( nshi )

// show services
for each s as NSSharingServiceMBS in ssl
  msgBox s.title
next
Notes:
items: The items to share.

Returns an array of sharing services to allow for items.
This method can be used to build a custom user interface or to populate a contextual menu.
Available in OS X v10.8 and later.
