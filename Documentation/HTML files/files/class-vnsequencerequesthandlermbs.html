class Vision MBS MacFrameworks Plugin 19.4 ✅ Yes ❌ No ❌ No ✅ Yes All
Function: An object that processes image analysis requests for each frame in a sequence.
Notes: Instantiate this handler to perform Vision requests on a series of images. Unlike the VNImageRequestHandlerMBS, you don't specify the image on creation. Instead, you supply each image frame one by one as you continue to call one of the perform methods.
