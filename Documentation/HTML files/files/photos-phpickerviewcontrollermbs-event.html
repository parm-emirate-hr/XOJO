PHPickerViewControllerMBS.didFinishPicking(Result() as PHPickerResultMBS)
Type Topic Plugin Version macOS Windows Linux iOS Targets
event Photos MBS iOS Plugin 22.5 ❌ No ❌ No ❌ No ✅ Yes
Function: Notifies the delegate that the user completed a selection or dismissed the picker using the cancel button.
Notes:
picker: The currently presented picker view controller.
results: The results of the user’s selections.

The system doesn’t automatically dismiss the picker after calling this method, but the plugin does.
