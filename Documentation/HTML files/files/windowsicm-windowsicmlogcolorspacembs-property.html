WindowsICMLogColorSpaceMBS.Filename as String
Type Topic Plugin Version macOS Windows Linux iOS Targets
property Windows ICM MBS Win Plugin 11.1 ❌ No ✅ Yes ❌ No ❌ No Desktop, Console & Web
Function: A string that names a color profile file.
Notes:
This member is typically set to "", but may be used to set the color space to be exactly as specified by the color profile. This is useful for devices that input color values for a specific printer, or when using an installable image color matcher. If a color profile is specified, all other members of this class should be set to reasonable values, even if the values are not completely accurate.
(Read and Write property)
