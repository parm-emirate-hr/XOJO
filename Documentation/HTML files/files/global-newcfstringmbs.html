Type Topic Plugin Version macOS Windows Linux iOS Targets
global method CoreFoundation MBS MacCF Plugin ✅ Yes ❌ No ❌ No ✅ Yes All
Function: Returns a CFStringMBS object created using the given string.
Notes:
Returns nil if s is empty.
The cfstring may be unicode.

See also NewCFStringMBS2 if you want to get an empty CFString object for an empty string.
