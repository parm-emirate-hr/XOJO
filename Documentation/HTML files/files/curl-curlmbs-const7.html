CURLMBS.kSSLOptionAllowBeast = 1
Type Topic Plugin Version
const CURL MBS CURL Plugin 20.3
Function: One of the SSL option flags.
Notes: tells libcurl to allow the BEAST SSL vulnerability in the name of improving interoperability with older servers. Some SSL libraries have introduced work-arounds for this flaw but those work-arounds sometimes make the SSL communication fail. To regain functionality with those broken servers, a user can this way allow the vulnerability back.
