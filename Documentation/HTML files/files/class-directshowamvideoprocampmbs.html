Function: The AMVideoProcAmp interface adjusts the qualities of an incoming video signal, such as brightness, contrast, hue, saturation, gamma, and sharpness.
Notes:
The WDM Video Capture filter exposes this interface if the hardware supports image adjustment.
This is an abstract class. You can't create an instance, but you can get one from various plugin functions.
