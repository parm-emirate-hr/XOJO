Type Topic Plugin Version
const DirectShow MBS Win Plugin 12.4
Function: One of the property selectors.
Notes:
Identifies the exposure setting, in log base 2 seconds. In other words, for values less than zero, the exposure time is 1/2n seconds, and for values zero or above, the exposure time is 2n seconds. For example:

DirectShowAMCameraControlMBS.kPropertyPan = 0
Type Topic Plugin Version
const DirectShow MBS Win Plugin 12.4
Function: One of the property selectors.
Notes: Identifies the camera's pan setting, in degrees. Values range from –180 to +180, with the default set to zero. Positive values are clockwise from the origin (the camera rotates clockwise when viewed from above), and negative values are counterclockwise from the origin.
DirectShowAMCameraControlMBS.kPropertyRoll = 2
Type Topic Plugin Version
const DirectShow MBS Win Plugin 12.4
Function: One of the property selectors.
Notes: Identifies the camera's roll setting, in degrees. Values range from –180 to +180, with the default set to zero. Positive values cause a clockwise rotation of the camera along the image-viewing axis, and negative values cause a counterclockwise rotation of the camera.
DirectShowAMCameraControlMBS.kPropertyTilt = 1
Type Topic Plugin Version
const DirectShow MBS Win Plugin 12.4
Function: One of the property selectors.
Notes: Identifies the camera's tilt setting, in degrees. Values range from –180 to +180, with the default set to zero. Positive values point the imaging plane up, and negative values point the imaging plane down.
DirectShowAMCameraControlMBS.kPropertyZoom = 3
Type Topic Plugin Version
const DirectShow MBS Win Plugin 12.4
Function: One of the property selectors.
Notes: Identifies the camera's zoom setting, in millimeters. Values range from 10 to 600, and the default is specific to the device.
