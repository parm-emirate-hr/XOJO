Type Topic Plugin Version macOS Windows Linux iOS Targets
method CoreFoundation Network MBS MacCF Plugin 3.3 ✅ Yes ❌ No ❌ No ✅ Yes All
Function: Adds authentication details.
Notes: Tries to modify request to contain the authentication information requested by authenticationFailureResponse (which presumably is a 401 or 407 response). Returns TRUE if successful; FALSE otherwise (leaving request unmodified). If authenticationScheme is NULL, the strongest supported scheme listed in failedResponse will be used.
