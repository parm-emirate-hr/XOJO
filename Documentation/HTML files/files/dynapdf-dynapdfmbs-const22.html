DynaPDFMBS.kif2UseProxy = 4
Type Topic Plugin Version
const DynaPDF MBS DynaPDF Plugin 11.3
Function: One of the constants for ImportFlags2.
Notes: Not meaningful for PDF files which are loaded from a memory buffer. If set, all streams are loaded from the file on demand but they are never hold in memory. This reduces drastically the memory usage and enables the processing of almost arbitrary large PDF files with minimal memory usage. The corresponding PDF file must not be deleted before CloseFile() or CloseFileEx() was called.
