Function: An alternative dictionary class for an ordered map with integer keys and string values.
Example: 
dim s as new IntegerToStringOrderedMapMBS

s.Value(1)="Hello"
s.Value(2)="World"

MsgBox str(s.Count) // shows 2

MsgBox s.Value(1)+" "+s.Value(2) // shows "Hello World"
Notes: You can choose whether you want to keep the map ordered using the OrderedMap class or whether you prefer a higher speed with the HashMap class.
